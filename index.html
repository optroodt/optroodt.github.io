Mega Man 2 Password Generator

This is a password generator for Mega Man 2. Click the bosses you want to have defeated and the energy tank to increase the number of energy tanks you'll start with.

I, Youri op 't Roodt, used to play Mega Man 2 and the other games from the series when I was a kid. The release of the NES Mini Classic edition, Analogue Nt mini and the RetroUSB AVS sparked my interest in NES games again. Mega Man 2 is included as one of the games on the NES Mini Classic.

The password scheme for all of the Mega Man games have long been documented, but I just wanted write my own implementation and create a nice user interface using JavaScript and React. If you want to read more about it, all the sources (including a Python implementation of the scheme) are available on GitHub.
